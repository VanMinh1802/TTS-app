|-------------------|----------------------|

#### Shared vs Feature Code

- **components/ui/**: Atomic components (Button, Input, Modal)
- **lib/**: Non-feature utilities (piper, audio, storage wrappers)
- **features/**: Feature-specific everything (api, components, hooks, types, store)
