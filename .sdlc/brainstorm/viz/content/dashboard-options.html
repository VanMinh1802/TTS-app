Dashboard — 3 hướng tiếp cận

Sub-project 1: Dashboard + Cards System

A

Incremental Polish

Giữ layout hiện tại, thêm animation + micro-interaction

  • Stats số chạy counter animate khi load
  • TiltCard hover elevation + glow
  • Progress bar animate fill + pulse edge
  • Bảng lịch sử có stagger row animation
  • Sửa spacing theo 8px rhythm

Pros

  • Nhanh (2h)
  • An toàn

Cons

  • Layout cũ
B

Layout Redesign

Bố cục mới tối ưu UX

  • Quick-actions row + welcome card
  • Stats grid 2x3 thay vì 1x3
  • Resource gauge kiểu ring/progress
  • Activity timeline thay bảng

Pros

  • Tươi mới
  • UX tốt

Cons

  • Tốn 5h
C

Widget-based Modular

Dashboard custom được per-user

  • Collapse/expand từng section
  • Drag-drop sắp xếp widget
  • Lưu layout per-user
  • Quá phức tạp cho version này

Pros

  • Premium

Cons

  • 12h
  • Over-engineer
