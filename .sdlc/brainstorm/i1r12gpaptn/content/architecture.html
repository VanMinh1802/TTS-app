Data Flow

Free User Flow

  1. Library page loads → useLibraryRecords reads from IndexedDB
  2. Cloud tab visible but disabled with "Pro feature" badge
  3. All CRUD operations hit IndexedDB only

Pro User Flow

  1. Library page loads → useLibraryRecords fetches from cloud API
  2. useLibrarySync kicks off background sync: local → cloud
  3. Each record gets status badge: Local / Cloud / Synced
  4. Deletes respect current tab context
